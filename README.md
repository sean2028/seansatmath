# Sean SAT Math Dashboard 2026

This folder is ready for GitHub Pages.

Files:
- index.html
- sat_function_form_dashboard.html
- sat_quadratics_conics_dashboard.html

GitHub Pages setup:
1. Create a public GitHub repository named satmath-2026.
2. Upload these three files to the repository root.
3. Go to Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main, folder: /root.
6. Save.
7. Wait 1–3 minutes for GitHub Pages to publish.

Your homepage will look like:
https://YOUR-GITHUB-USERNAME.github.io/satmath-2026/

Then add this link to Google Sites as a Button or Embed by URL.
