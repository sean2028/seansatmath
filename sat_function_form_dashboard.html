Quick Strategy: The “See It or Calculate It?” Test

Golden rule: These SAT questions are not only asking you to calculate a value. They ask whether that value is already displayed as one visible constant or one visible coefficient.
Step 1

Identify the target value

Circle the exact phrase in the question:

  • “y-coordinate of the y-intercept”
  • “minimum value”
  • “maximum value”
  • “f(1) = k”
  • “displays as a constant or coefficient”
Step 2

Use the anchor input

TargetAnchor input
y-interceptx = 0
minimum/maximum with x ≥ 0usually endpoint x = 0
f(1) = kx = 1
Step 3

Ask the display question

After finding the value, ask:

Is this exact whole value visible as one constant or one coefficient?

If you still need to add, multiply, substitute, or rewrite, then it is hidden.

Type 1

Minimum / Maximum Value Displayed as a Coefficient

Question pattern: “Which equation displays, as a constant or coefficient, the minimum/maximum value of the function it defines, where x ≥ 0?”

Core idea

For an exponential function:

A(b)^x
BaseBehaviorEndpoint value on x ≥ 0
b > 1IncreasingMinimum occurs at x = 0
0 < b < 1DecreasingMaximum occurs at x = 0

Display test

Equation formValue at x = 0Does the equation display it?
A(b)^xAYes
A(b)^(x+c)A(b)^cNo, unless simplified
A(b)^x + CA + CNo, unless A + C is shown as one number

Apply to Sean’s Q16

Visible equation II:

g(x) = 18(1.16)(1.4)^(x+3)
  1. The base is 1.4, and 1.4 > 1. So the function is increasing.
  2. The domain is x ≥ 0, so the minimum happens at the left endpoint, x = 0.
  3. Plug in x = 0:
g(0) = 18(1.16)(1.4)^(0+3)
g(0) = 18(1.16)(1.4)^3

The minimum is 18(1.16)(1.4)^3, but the equation does not show that as one coefficient. It shows the pieces separately and has a shifted exponent x+3.

Conclusion for Q16: Equation II does not display the minimum value directly. A form like M(1.4)^x would display the minimum as coefficient M.
Type 2

Y-Intercept Hidden Inside Equivalent Exponential Forms

Question pattern: Original function plus equivalent forms. The question asks which form displays the y-coordinate of the y-intercept.

Core idea

y-intercept = function value when x = 0

Then you must check whether that whole value is visible as one constant or coefficient.

Apply to Sean’s Q19

Original function:

f(x) = a(3.1^x + 3.1^b)

Plug in x = 0:

f(0) = a(3.1^0 + 3.1^b)
f(0) = a(1 + 3.1^b)

So the y-intercept is:

a(1 + 3.1^b)

Check equation I

g(x) = a(3.1^x + k)

Because this is equivalent to the original function:

k = 3.1^b

Now plug in x = 0:

g(0) = a(1 + k)

The equation shows a and k, but it does not show a(1+k) as one constant or coefficient.

Check equation II

h(x) = a(3.1)^x + m

Equivalent form means:

m = a(3.1^b)

Plug in x = 0:

h(0) = a + m

The equation shows a and m, but it does not show a+m as one constant or coefficient.

Conclusion for Q19: Neither I nor II displays the y-intercept directly.
Trap: An equivalent equation can still fail the “display” test. Equivalent means same function, not necessarily best form.
Type 3

Shifted Exponent vs Outside Constant

Question pattern: Two functions are given. Decide whether either equation displays the y-intercept as a constant or coefficient.

Apply to Sean’s Q21

Equation I

f(x) = a(3.3)^(x+b)

Plug in x = 0:

f(0) = a(3.3)^(0+b)
f(0) = a(3.3)^b

The equation only displays coefficient a. But the y-intercept is a(3.3)^b. So equation I does not display it directly.

To see the hidden coefficient, rewrite:

a(3.3)^(x+b) = a(3.3)^b(3.3)^x

After rewriting, the coefficient a(3.3)^b is visible. But in the original equation, it was hidden.

Equation II

g(x) = a(3.3)^x + b

Plug in x = 0:

g(0) = a(3.3)^0 + b
g(0) = a + b

The equation displays a and b separately, but it does not display a+b as one constant or coefficient.

Conclusion for Q21: Neither I nor II displays the y-intercept directly.

Master Trap Table

Form Common wrong thought Correct y-intercept Display status
A(b)^x Coefficient is y-intercept A Displayed
A(b)^x + C Constant C is y-intercept A + C Usually hidden
A(b)^(x+c) Coefficient A is y-intercept A(b)^c Hidden unless rewritten
A(b^x + D) Inside constant D helps directly A(1+D) Hidden
A(b)^x + D D is y-intercept A + D Hidden unless combined
M(b)^x, x ≥ 0, b > 1 Minimum may require work M Minimum displayed
M(b)^(x+c), x ≥ 0, b > 1 M is minimum M(b)^c Hidden
Memory phrase: If exponent is exactly x, the coefficient is easy to trust. If exponent is x + something, the coefficient is hiding another factor.

Practice Bank: 2 Questions for Each Type

Type 1 Practice — Minimum / Maximum

Practice 1A: Minimum value

For x ≥ 0, which equation displays, as a coefficient, the minimum value of the function it defines?

I. f(x) = 45(1.2)^x

II. g(x) = 45(1.2)^(x+2)

A. I only   B. II only   C. I and II   D. Neither

Answer: A. Since 1.2 > 1, minimum occurs at x=0. I gives 45 directly. II gives 45(1.2)^2, not displayed directly.
Practice 1B: Maximum value

For x ≥ 0, which equation displays, as a coefficient, the maximum value of the function it defines?

I. p(x) = 80(0.5)^x

II. q(x) = 20(0.5)^(x-2)

A. I only   B. II only   C. I and II   D. Neither

Answer: A. Since 0.5 < 1, the function decreases, so maximum occurs at x=0. I displays 80. II has maximum 20(0.5)^(-2)=80, but 80 is not displayed directly.

Type 2 Practice — Equivalent Exponential Forms

Practice 2A

The function f is defined by f(x)=c(2^x+2^d). Equivalent forms are:

I. g(x)=c(2^x+r)

II. h(x)=c(2)^x+s

Which equation displays the y-coordinate of the y-intercept as a constant or coefficient?

A. I only   B. II only   C. I and II   D. Neither

Answer: D. The y-intercept is c(1+2^d). I gives c(1+r); II gives c+s. Neither displays the whole value directly.
Practice 2B

The function f is defined by f(x)=m(5^x+7). Equivalent forms are:

I. g(x)=m(5^x+k)

II. h(x)=m(5)^x+n

Which equation displays the y-coordinate of the y-intercept?

A. I only   B. II only   C. I and II   D. Neither

Answer: D. The y-intercept is m(1+7)=8m. Neither equation shows 8m as one constant or coefficient.

Type 3 Practice — Shifted Exponent / Outside Constant

Practice 3A

I. f(x)=12(3)^(x+2)

II. g(x)=12(3)^x+2

Which equation displays the y-coordinate of the y-intercept as a constant or coefficient?

A. I only   B. II only   C. I and II   D. Neither

Answer: D. I has y-intercept 12(3)^2=108, not displayed. II has y-intercept 12+2=14, not displayed as one constant/coefficient.
Practice 3B

I. f(x)=75(4)^x

II. g(x)=15(4)^(x+1)

Which equation displays the y-coordinate of the y-intercept as a constant or coefficient?

A. I only   B. II only   C. I and II   D. Neither

Answer: A. I has y-intercept 75 and displays it. II has y-intercept 15(4)=60, but displays coefficient 15.

Sean’s 20-Second Exam Routine

  1. Circle the target phrase. Is the question asking for y-intercept, minimum, maximum, or a specific function value?
  2. Choose the anchor input. Y-intercept uses x=0. Endpoint minimum/maximum usually uses x=0 when domain is x≥0.
  3. Compute the exact target value. Keep it symbolic if needed, like a(1+3.1^b).
  4. Compare to the equation. Is that whole value already visible as one constant or coefficient?
  5. Do not be fooled by equivalent forms. Equivalent functions may still hide the requested value.
