1. Standard Quadratic Form

y = ax² + bx + c

This is the most common form on the SAT.

Opening

a > 0: parabola opens up.

a < 0: parabola opens down.

Intercept Shortcut

The y-intercept is (0, c), because when x = 0, y = c.

Very useful SAT shortcut: a + b + c = f(1), because f(1) = a(1)² + b(1) + c.

2. Vertex Formula from Standard Form

For:

y = ax² + bx + c

The x-coordinate of the vertex is:

x = -b / (2a)

Then plug that x-value back into the equation to get the y-coordinate.

Vertex = ( -b/(2a), f(-b/(2a)) )

Example

y = x² - 6x + 5

Here a = 1 and b = -6.

x = -(-6)/(2 · 1) = 3
y = 3² - 6(3) + 5 = -4

Vertex:

(3, -4)

3. Vertex Form

y = a(x - h)² + k

The vertex is:

(h, k)

This is the best form when the problem gives a vertex.

Sign Warning

(x - 3)² means the vertex x-value is 3, not -3.

Sign Warning

(x + 4)² means the vertex x-value is -4.

Example

y = 2(x - 3)² - 5

Vertex:

(3, -5)

Another Example

y = a(x + 4)² + 7

Vertex:

(-4, 7)

4. Factored Form / Intercept Form

y = a(x - r₁)(x - r₂)

The x-intercepts are:

(r₁, 0) and (r₂, 0)

Example

y = 2(x - 1)(x - 5)

The x-intercepts are:

(1, 0) and (5, 0)

The axis of symmetry is halfway between the roots:

x = (r₁ + r₂) / 2

For this example:

x = (1 + 5) / 2 = 3

5. Quadratic Formula

Use this when you need to solve:

ax² + bx + c = 0
x = [-b ± √(b² - 4ac)] / (2a)

This gives the x-intercepts of the parabola.

6. Discriminant

b² - 4ac

It tells you how many real x-intercepts the parabola has.

DiscriminantMeaning
b² - 4ac > 0Two real solutions; the parabola crosses the x-axis twice.
b² - 4ac = 0One real solution; the parabola touches the x-axis at the vertex.
b² - 4ac < 0No real solutions; the parabola does not touch the x-axis.

7. Completing the Square

This converts standard form into vertex form.

Example

y = x² - 8x + 3

Group the x-terms:

y = (x² - 8x) + 3

Take half of -8, then square it:

-8 / 2 = -4, and (-4)² = 16

Add and subtract 16:

y = (x² - 8x + 16) - 16 + 3
y = (x - 4)² - 13

So the vertex is:

(4, -13)
SAT often tests completing the square because it reveals the vertex.

8. Maximum and Minimum Value

For:

y = a(x - h)² + k
a > 0

The parabola opens up, so the vertex is a minimum.

Minimum value = k.

a < 0

The parabola opens down, so the vertex is a maximum.

Maximum value = k.

Example

y = -3(x - 2)² + 10

Since a = -3 < 0, it opens downward.

Maximum value:

10

Vertex:

(2, 10)

9. Axis of Symmetry

FormAxis of symmetry
y = ax² + bx + cx = -b / (2a)
y = a(x - h)² + kx = h
y = a(x - r₁)(x - r₂)x = (r₁ + r₂) / 2
This is very useful when the SAT gives two roots.

10. Circle Equation

(x - h)² + (y - k)² = r²

Center:

(h, k)

Radius:

r

Example

(x - 3)² + (y + 2)² = 25

Center:

(3, -2)

Radius:

5
(y + 2)² means the center’s y-coordinate is -2.

11. Ellipse Equation

((x - h)² / a²) + ((y - k)² / b²) = 1

Center:

(h, k)

The larger denominator tells you the longer direction.

Example

((x - 2)² / 25) + ((y + 1)² / 9) = 1

Center:

(2, -1)

Since 25 > 9, the ellipse is wider horizontally.

Horizontal radius = 5, Vertical radius = 3
On the SAT, you usually only need center, radius lengths, and basic graph interpretation.

12. Hyperbola Equation

Less common on the SAT, but still possible.

Horizontal hyperbola

((x - h)² / a²) - ((y - k)² / b²) = 1

Vertical hyperbola

((y - k)² / a²) - ((x - h)² / b²) = 1

Center:

(h, k)
Circle and ellipse have plus between the squared terms. Hyperbola has minus between the squared terms.

14. Best SAT Decision Strategy

When you see a quadratic question, first ask: What information is given?

  • If given vertex, use y = a(x - h)² + k
  • If given roots / x-intercepts, use y = a(x - r₁)(x - r₂)
  • If given a, b, c and asked for vertex, use x = -b/(2a)
  • If asked about number of solutions / intersections, use b² - 4ac
  • If asked about a + b + c, think a + b + c = f(1)

Q26 Pattern Recap

Problem type: vertex given, asks for a + b + c.

  1. Vertex is (9, -14), so use vertex form:
f(x) = a(x - 9)² - 14
  1. The parabola intersects the x-axis at two points and the vertex is below the x-axis, so it opens upward.
a > 0
  1. Recognize the shortcut:
a + b + c = f(1)
  1. Plug x = 1 into vertex form:
f(1) = a(1 - 9)² - 14 = 64a - 14
  1. Since a > 0:
64a - 14 > -14

So a + b + c must be greater than -14. Among -23, -19, -14, and -12, only -12 works.

Answer: D, -12
