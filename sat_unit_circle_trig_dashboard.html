Core unit-circle idea

Terminal point

On the unit circle, the terminal point is always:

$(x,y)=(\cos\theta,\sin\theta)$
FunctionUnit-circle meaning
$\cos\theta$$x$-coordinate
$\sin\theta$$y$-coordinate
$\tan\theta$$\dfrac{y}{x}=\dfrac{\sin\theta}{\cos\theta}$

Terminal side / coterminal angles

  • Angles start on the positive $x$-axis.
  • Positive angles rotate counterclockwise.
  • Negative angles rotate clockwise.
  • Coterminal angles end on the same terminal side.
$\theta \pm 360^\circ$   or   $\theta \pm 2\pi$

SAT solving flow

  1. Make the angle coterminal if needed.
  2. Find the quadrant.
  3. Find the reference angle.
  4. Use the special-angle value.
  5. Apply the sign from the quadrant.

Useful identities and SAT question types

Pythagorean identity

$\sin^2\theta+\cos^2\theta=1$

Use this when one trig value is given and another is needed. Then use the quadrant to choose the sign.

Complementary angles

If $A+B=90^\circ$, then:

$\sin A=\cos B$
$\cos A=\sin B$

Common SAT trig question types

  • Find trig values from a point $(x,y)$ on the terminal side.
  • Determine sign from quadrant.
  • Use right-triangle ratios.
  • Use complementary angles.
  • Convert degrees and radians.

Mini practice bank

1. Find $\sin 240^\circ$, $\cos 240^\circ$, and $\tan 240^\circ$.
$240^\circ$ is QIII. Reference angle $=60^\circ$. So $\sin=-\sqrt3/2$, $\cos=-1/2$, $\tan=\sqrt3$.
2. The terminal side passes through $(-8,15)$. Find $\tan\theta$.
$\tan\theta=y/x=15/(-8)=-15/8$.
3. If $A+B=90^\circ$ and $\sin A=\sqrt{15}/4$, find $\cos B$.
Complementary identity: $\sin A=\cos B$, so $\cos B=\sqrt{15}/4$.
4. If $\cos\theta=-4/5$ and $\theta$ is in QII, find $\sin\theta$.
$\sin^2+\cos^2=1$. $\sin^2=1-16/25=9/25$. QII sine is positive, so $\sin\theta=3/5$.
